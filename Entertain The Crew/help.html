ABOUT THE MODES:

  • Alien v/s Astronaut -> human would be playing versus AI player i.e, against computer player
  • Astronaut v/s Astronaut -> 2 players can play against each other

Alien vs Astronaut mode there are 3 levels : EASY MEDIUM HARD
    Astronaut vs Astronaut mode there are 3 modes:
  • ON MARS -> 2 human players can play the game against each other on mars
  • ON EARTH -> 2 human players can play the game on Earth where GRAVITY influences the game as it makes the 0/X fall to the bottom
  • + DRAGON -> 2 astronauts play against each other but the dragon attacks the game with fire which makes few 0/X to get disappeared

Keyboard Accessibility

Game can be played either using the keyboard or with the cursor
User can traverse through the options using right arrow key and the game can be played using the number keys
Or he/she can play using cursor by clicking on the screen.
